# Dotfiles

My personal configuration files for Arch Linux.

## Stack
- WM: Qtile
- OS: Arch Linux
- Compositor: Picom
- Terminal: Alacritty
- Editor: Vim (NERDTree, CoC, Airline, Copilot, Discord RPC)

## Qtile Features
- Multiple layouts: MonadTall, MonadWide, Matrix, RatioTile
- Custom bar with battery, volume, clock and systray
- Floating window animations
- Auto wallpaper on startup
- Vim-style keybindings (hjkl)

## Vim Features
- Autocompletion via CoC
- Async linting via ALE (ESLint, clangd)
- File explorer via NERDTree
- Laravel Blade syntax support
- GitHub Copilot
- Discord Rich Presence
